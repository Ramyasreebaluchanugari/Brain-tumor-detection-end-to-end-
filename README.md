# Brain Tumor Detection (End-to-End)

## Overview
This project is a Brain Tumor Detection system that uses Deep Learning to classify MRI brain images as tumor or non-tumor. The application is built with Python, TensorFlow/Keras, Flask, HTML, and CSS.

## Features
- Upload MRI brain images
- Predict whether a brain tumor is present
- User-friendly web interface
- Deep learning model for image classification

## Technologies Used
- Python
- TensorFlow / Keras
- Flask
- HTML
- CSS

## Project Structure
```
BrainTumorDetection/
│── app.py
│── train.py
│── predict.py
│── requirements.txt
│── templates/
│── static/
│── README.md
```

## Dataset
The dataset was obtained from Kaggle and is not included in this repository due to its large size.

## How to Run
1. Clone the repository.
2. Install the required packages:
   ```
   pip install -r requirements.txt
   ```
3. Run the application:
   ```
   python app.py
   ```
4. Open the browser and visit:
   ```
   http://127.0.0.1:5000
   ```
